# Frontend Guidelines: Apex Blog

## 1. Core Principles
- [cite_start]**Clarity**: AI summaries must be scannable.
- **Accessibility**: WCAG 2.1 AA compliance.
- **Responsive**: Mobile-first grid layouts.

## 2. Design Tokens
- **Primary Blue**: #3b82f6
- **Slate Neutral**: #64748b
- **Success**: #22c55e | **Error**: #ef4444
- **Spacing**: 4px increments (base-4).

## 3. Components
- **Buttons**: Rounded-md, blue-600 background, white text.
- **Cards**: Border-slate-200, shadow-sm, hover effect.
- **Inputs**: Ring-blue-500 on focus.

## 4. Breakpoints
- sm: 640px
- md: 768px
- lg: 1024px
- xl: 1280px
